The Game

Objective
▪ Collect points in the tournaments
▪ The team with the most points at the end wins

Tournament
▪ In each tournament, all teams compete in a match with 4 lines
▪ The winning teams (longest line) play a match against each other

© itestra 2026

The Game
End of the game:
If one of the following scenarios occurs, the game ends immediately:
▪ No snakes are left
▪ Exactly one snake is still alive
▪ If the game stalls (e.g., two snakes are running in circles, etc.)

Scoring
▪ The base points per round correspond to the length of your snake at the end of a game
▪ The snake with the most points wins
▪ The snake that survives the longest gets a bonus, but does not automatically win
▪ Later games award extra points
• Grand Finale (Sunday 10:00)

How do you play the game?
Client

Server
▪ The game server ticks once per second* at fixed intervals
▪ The game field is updated per tick
▪ The current game field can be requested from the server

Clients
▪ You query the game board once per tick
▪ You calculate your next move
▪ You post the direction in which your snake should move in the next tick
north, west, south, or east

Game Tick in Detail 1/2
Reading the direction
▪ For each snake, the most recently posted direction is evaluated

Movement Phase
▪ All snakes move one space simultaneously in the set direction
▪ The snake can leave the playing field in any direction; it re-enters at the opposite edge

Collisions
▪ If a snake's head touches any object:
• The snake dies
▪ All collisions are handled simultaneously

© itestra 2026

Game Tick in Detail 2/2
Collecting Items
▪ Any item touched by a snake's head is collected
▪ Apples cause the snake to grow one space immediately
• The last tile of the snake is duplicated

New items spawn
▪ After all the snake's actions have been processed, new items spawn
▪ Items spawn randomly, but not on already occupied spaces.
